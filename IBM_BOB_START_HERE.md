# Start Here in IBM Bob

## 1. Open the project

Extract the ZIP and open the folder `AI-Phishing-Email-Detection-IBM-Bob` in IBM Bob IDE.

## 2. Initialize Bob

Open the agentic chat, switch to **Code mode**, and run:

```text
/init
```

Approve repository read/write access when prompted. The project already includes `AGENTS.md`; ask Bob to read and follow it.

## 3. Give Bob this exact first task

```text
Read AGENTS.md and inspect the complete repository. Set up the Python environment, install requirements, train the model, run pytest -q, and start the FastAPI application. Fix only genuine errors and keep all safety limitations. Tell me the dashboard URL and API documentation URL when complete.
```

## 4. Open the project

- Dashboard: `http://127.0.0.1:8000`
- API documentation: `http://127.0.0.1:8000/docs`
