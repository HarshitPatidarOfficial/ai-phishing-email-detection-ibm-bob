# Project Submission Checklist

## Required evidence

- [ ] Working public GitHub repository link
- [ ] README with problem statement, architecture, setup, API, and limitations
- [ ] At least three project screenshots from `docs/screenshots/`
- [ ] Passing test output screenshot
- [ ] IBM Bob workspace screenshot showing the project open
- [ ] IBM Bob chat screenshot showing `/init` or a completed project task
- [ ] IBM Bob certificate or badge obtained from the official IBM learning/event process
- [ ] Short demonstration video link (optional but recommended)

## Important

A certificate must be earned or issued by IBM. This repository deliberately does not generate or imitate a certificate. Add only authentic evidence to the submission.
